# 临界指数与标度关系

## 一、临界指数定义

- 比热：\( C_\rho \sim |T - T_c|^{-\alpha} \)
- 密度差：\(\Delta\rho \sim (T_c - T)^{\beta}\)
- 磁化率：\(\chi_2 \sim (T - T_c)^{-\gamma}\)
- 临界点密度与化学势：\(\rho - \rho_c \sim (\mu - \mu_c)^{1/\delta}\)

此外定义：
\[
\chi_2^B \sim |\mu - \mu_0|^{-\nu}
\]
\[
S\sigma \sim |\mu - \mu_0|^{-\zeta}
\]
\[
\kappa\sigma^2 \sim |\mu - \mu_0|^{-\eta}
\]

## 二、标度律（PNJL模型平均场近似下成立）

\[
\alpha + 2\beta + \gamma = 2 \quad (\text{Rushbrooke标度律})
\]
\[
\gamma = \beta(\delta - 1) \quad (\text{Widom标度律})
\]

## 三、旋节线边界临界指数

\[
\chi_2^B \sim |\mu - \mu_0|^{-\nu'}
\]
\[
S\sigma \sim |\mu - \mu_0|^{-\zeta'}
\]
\[
\kappa\sigma^2 \sim |\mu - \mu_0|^{-\eta'}
\]

其中 \(\mu_0\) 为旋节点化学势。
