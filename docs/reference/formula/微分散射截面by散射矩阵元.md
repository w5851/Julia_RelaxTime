# 微分散射截面公式

在相对论性 Boltzmann 动力学理论中，微分散射截面与散射矩阵元之间的关系由以下公式给出：

\[
\frac{d\sigma}{dt} = \frac{1}{16\pi s_{ij}^+ s_{ij}^-} \cdot \frac{1}{4N_C^2} \sum_{\text{spin, color}} |\mathcal{M}|^2
\]

## 符号说明

- \(\displaystyle \frac{d\sigma}{dt}\)：微分散射截面，表示在 Mandelstam 变量 \(t\) 方向上的散射概率密度。
- \(s_{ij}^\pm\)：定义为 \(s - (m_i \pm m_j)^2\)，其中 \(s\) 是 Mandelstam 变量 \(s\)，\(m_i, m_j\) 是入射粒子的质量。
- \(N_C\)：色数，通常取值为 3。
- \(\displaystyle \sum_{\text{spin, color}}\)：对所有自旋和颜色态求和并对初始态求平均。
- \(|\mathcal{M}|^2\)：散射矩阵元 \(\mathcal{M}\) 的模平方，反映了散射过程的振幅强度。

## 说明

该公式将微分散射截面与散射过程的动力学信息（通过散射矩阵元 \(\mathcal{M}\) 体现）联系起来，是计算总散射截面的基础。散射矩阵元 \(\mathcal{M}\) 的具体形式依赖于所考虑的相互作用模型和散射过程。
