# Mott 相变位置计算

本文给出 PNJL 模型中 Mott 相变线的判断条件与计算步骤，供相结构与输运分析使用。

## 1. 物理定义
当介子质量 $M_M$ 等于其组分夸克动力学质量之和时，介子由束缚态转为共振态，可衰变为夸克-反夸克对：

$$
M_M(T, \mu) = M_{q_1}(T, \mu) + M_{q_2}(T, \mu)
$$

该条件满足的 $(T,\mu)$ 点构成 Mott 相变线。

## 2. 常见介子阈值条件
- π 介子：$M_{\pi} = M_u + M_d$
- K 介子：$M_{K} = M_u + M_s$

## 3. 计算步骤
1. 设定温度 $T$ 与重子化学势 $\mu_B$（或夸克化学势 $\mu_q$）扫描范围。
2. 解平均场方程得到 $M_u, M_d, M_s$。
3. 求解介子质量 $M_\pi(T,\mu)$、$M_K(T,\mu)$（见 [MesonMass_RPA_Pole.md](MesonMass_RPA_Pole.md)）。
4. 计算差值：
   - $\Delta_\pi = M_\pi - (M_u + M_d)$
   - $\Delta_K = M_K - (M_u + M_s)$
5. 满足 $\Delta=0$ 的点即为 Mott 相变点，连接后得到相变线。

## 4. 相变关系与数值特征
- 手征相变导致 $M_f$ 快速下降，Mott 相变通常出现在手征平滑过渡附近或略高温区。
- 退禁闭相变由 Polyakov loop $\Phi$ 表征，Mott 线温度可能高于退禁闭温度。
- 临界终点（CEP）附近，Mott 线可能绕过 CEP 进入高密度区域。
- 温度升高时 $M_M$ 通常升高而 $M_{q_1}+M_{q_2}$ 降低，两者交点为 Mott 点。

## 5. 注意事项
- 当 $M_M$ 超过阈值时，传播子出现虚部（宽度 $\Gamma>0$），介子变为共振态。
- 平均场近似可能低估涨落效应，CEP 附近需谨慎解释。
- 介子质量与夸克质量应自洽迭代求解以保证收敛。
