专家报告：Soloveva等人(2021)文章中方程(38)的推导与物理内涵解析本文旨在对Soloveva等人(2021)发表在《物理评论C》上的文章“Shear viscosity and electric conductivity of a hot and dense QGP with a chiral phase transition” 1 中用于夸克物质散射截面计算的核心公式——方程(38)——进行详尽的推导解析和物理意义阐释。方程(38)给出了夸克-夸克（$qq$）或夸克-反夸克（$q\bar{q}$）散射的微分截面，是计算夸克胶子等离子体（QGP）输运系数（如剪切粘滞系数$\eta$和电导率$\sigma_0$）的关键输入量。$$\frac{d\sigma}{dt}=\frac{1}{16\pi s_{12}^{+}s_{12}^{-}}\frac{1}{4N_{C}^{2}}\sum_{s,c}|\mathcal{M}_{s/u}-\mathcal{M}_{t}|^{2}.\quad \text{(Eq. 38)}$$I. 相对论 $2 \to 2$ 散射理论的基础原理方程(38)的结构源于量子场论（QFT）中对二体散射过程 $1+2 \to 3+4$ 的相对论描述。微分截面$d\sigma/dt$的计算依赖于两个核心要素：洛伦兹不变相位空间因子（LIPS）和相互作用的散射矩阵元平方$|\mathcal{M}|^2$ 2。I.A. 相对论运动学与Mandelstam变量在相对论散射理论中，过程的运动学特性通常由Mandelstam变量$s, t, u$描述，这些变量基于粒子的四动量$p_i$定义 4。$s$ 变量（质心能量平方）： $s = (p_1 + p_2)^2$，表示入射粒子四动量之和的平方，即质心系（CM）的总能量平方 4。它决定了反应中可用的总能量，并设置了反应的运动学阈值。$t$ 变量（四动量转移平方）： $t = (p_1 - p_3)^2$，表示入射粒子1与出射粒子3之间四动量转移的平方 4。在质心系中，$t$与散射角$\theta_{cm}$直接相关，因此$d\sigma/dt$即是微分截面相对于四动量转移的表示。$u$ 变量： $u = (p_1 - p_4)^2$，是第三个Mandelstam变量。这三个变量并非独立，它们受限于质量壳条件：$$s + t + u = m_1^2 + m_2^2 + m_3^2 + m_4^2$$4。I.B. 从QFT到微分截面的转换根据QFT原理，微分截面$d\sigma$与矩阵元平方$|\mathcal{M}|^2$的关系为：$$d\sigma = \frac{1}{\text{Flux}} |\mathcal{M}|^2 d\text{LIPS}_2$$其中，$\text{Flux}$是入射通量因子，$d\text{LIPS}_2$是二体终态的洛伦兹不变相位空间元 2。通过在质心系中进行计算，并利用动量守恒和能量守恒的$\delta$函数对LIPS进行积分，可以将$d\sigma/d\Omega_{cm}$（相对于质心角）转换为$d\sigma/dt$。最终的标准相对论公式为：$$\frac{d\sigma}{dt} = \frac{1}{64\pi s p_{cm}^2} |\mathcal{M}|^2$$其中$p_{cm}^2$是质心系中初始粒子的三动量平方。Soloveva等人的研究中也提到了这一标准形式的积分关系，例如公式(49)中包含的因子$\frac{1}{64\pi s p_{cm}^2}$ 1。II. 方程(38)的解构：运动学因子与归一化方程(38)是上述标准相对论公式在PNJL模型语境下的具体表达。II.A. 运动学分母 $\frac{1}{16\pi s_{12}^{+}s_{12}^{-}}$ 的详细分析方程(38)中的分母项$\frac{1}{16\pi s_{12}^{+}s_{12}^{-}}$代表了QFT推导中所需的运动学因子$\frac{1}{64\pi s p_{cm}^2}$ 1。这一等价性基于夸克物质研究中常用的Källén函数（或三角形函数）$\lambda(a, b, c)$。PNJL符号的定义： 在PNJL模型中，引入了简化的符号$s_{ij}^{\pm}$，定义为 [1 (Eq. 34)]:$$s_{ij}^{\pm} = s-(m_{i}\pm m_{j})^{2}$$对于初始粒子1和2，其乘积$s_{12}^{+}s_{12}^{-}$恰好等于Källén函数$\lambda(s, m_1^2, m_2^2)$：$$s_{12}^{+}s_{12}^{-} = [s - (m_1 + m_2)^2][s - (m_1 - m_2)^2] = \lambda(s, m_1^2, m_2^2)$$这个乘积项在$s$低于运动学阈值$(m_1+m_2)^2$时为负，这恰当地反映了相位空间消失的物理条件。与质心动量的关联： 初始粒子的质心系动量平方$p_{cm}^2$定义为 [1 (Eq. 51)]:$$p_{cm}^2 = \frac{\lambda(s, m_1^2, m_2^2)}{4s} = \frac{s_{12}^{+}s_{12}^{-}}{4s}$$运动学因子的验证： 将$s_{12}^{+}s_{12}^{-}$代入标准微分截面公式的分母：$$\frac{1}{64\pi s p_{cm}^2} = \frac{1}{64\pi s \cdot \frac{s_{12}^{+}s_{12}^{-}}{4s}} = \frac{1}{16\pi s_{12}^{+}s_{12}^{-}}$$因此，方程(38)的第一项$\frac{1}{16\pi s_{12}^{+}s_{12}^{-}}$是标准的相对论运动学因子$\frac{1}{64\pi s p_{cm}^2}$的精确重写，确保了计算出的微分截面正确地遵守了能量和动量守恒所施加的运动学约束。这种使用$s_{ij}^{\pm}$的符号约定在PNJL/NJL模型中尤其方便，因为它简化了散射矩阵元平方（例如公式(29)到(37)）中自然出现的代数项的表达 1。II.B. 归一化因子 $\frac{1}{4N_{C}^{2}}$微分截面必须对入射粒子的内部自由度进行平均。因子$\frac{1}{4N_{C}^{2}}$来源于对初始态的自旋和颜色自由度的平均：自旋平均： 对于两个入射费米子（夸克或反夸克），每个粒子有2个自旋态。入射自旋平均因子为$\frac{1}{2 \cdot 2} = \frac{1}{4}$。颜色平均： 对于$N_C=3$色的QCD，每个入射夸克有$N_C=3$个颜色态。入射颜色平均因子为$\frac{1}{N_C \cdot N_C} = \frac{1}{3^2} = \frac{1}{9}$。总归一化： 总平均因子为$\frac{1}{4} \times \frac{1}{N_C^2} = \frac{1}{4N_C^2}$。矩阵元平方项$\sum_{s,c}|\mathcal{M}{s/u}-\mathcal{M}{t}|^{2}$则表示对终态所有自旋$s$和颜色$c$态的求和。综合这些因素，方程(38)精确地表示了非极化的$2 \to 2$散射微分截面。III. 微观模型：PNJL框架下的夸克相互作用方程(38)中的核心项是色平均、自旋求和的矩阵元平方$\sum_{s,c}|\mathcal{M}{s/u}-\mathcal{M}{t}|^{2}$。这一项完全由计算所采用的有效场论模型——扩展$N_f=3$ Polyakov Nambu-Jona-Lasinio (PNJL) 模型——所决定 1。III.A.
