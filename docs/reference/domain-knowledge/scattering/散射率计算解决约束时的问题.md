你问的非常关键，而且是做散射率计算时最容易困惑的点之一。

# 2. 两体相空间（two-body phase space）是标准结论

[
\int \frac{d^3p_3}{(2\pi)^3 2E_3}

(2\pi)^4\delta^4(p_1+p_2-p_3-p_4)

它告诉你：

> **把实验室系里对 (d^3p_3 d^3p_4) 的六维积分替换为质心系的角积分后，
> 所有雅可比都已经“内置”在相空间因子 (\frac{p^*}{8\pi\sqrt{s}}) 里了。**

也就是说，这个公式是一个经过完整的变量替换和 δ 函数处理后得到的普适结果。

# 3. 为什么 Lorentz boost 不需要雅可比？

答案是：

### ★ 你在 CM 系上积分角变量 **并没有把积分变量改成实验室系变量**。

你只是用 boost 来“查询”实验室系的量，例如：

* 三动量的方向
* 实验室系下的 (|\mathbf p|)、(\cos\theta_{\rm lab})
* 各向异性分布 (f(p))

积分 measure 仍然是 **(d\Omega^*=d\cos\theta^*,d\phi^*)**。

因此不需要任何 boost 的雅可比。

你不是把变量从 CM 变到 Lab；
你是在 CM 变量上积分，但需要在 Lab 中评估被积函数。
这是完全不需要雅可比的操作。

[
\int \frac{d^3\mathbf p_3}{2E_3}\frac{d^3\mathbf p_4}{2E_4},(2\pi)^4\delta^{(4)}(P-p_3-p_4)
]
化简成含有质心系角积分 (\propto p^*/\sqrt{s},\mathrm d\Omega^*) 的形式，并说明在常见的 S 矩阵/相空间约定下得到的结果是
[
\frac{p^*}{8\pi\sqrt{s}},\mathrm d\Omega^*.
]

---

## 约定与目标

* 用 (P^\mu=(P^0,\mathbf P)) 表示总四动量，定义 (s=P^2)（通常 (P^0=\sqrt{s}) 在质心系中）。
* (p_3^\mu=(E_3,\mathbf p_3),; p_4^\mu=(E_4,\mathbf p_4))，并且 (E_i=\sqrt{\mathbf p_i^2+m_i^2})。
* 我们要化简的对象（把所有 ((2\pi)) 因子显式写出）是典型的两体相空间元素
  [
  d\Phi_2 \equiv (2\pi)^4\delta^{(4)}(P-p_3-p_4),\frac{d^3\mathbf p_3}{(2\pi)^3 2E_3},\frac{d^3\mathbf p_4}{(2\pi)^3 2E_4}.
  ]
  （这是粒子物理中 S 矩阵/微分截面常用的约定。）

---

## 第一步：用空间 δ 消去一个三维积分

把空间部分的 δ 函数用于 (d^3\mathbf p_4)：
[
d\Phi_2
= \frac{1}{(2\pi)^2}\int \frac{d^3\mathbf p_3}{4E_3E_4},\delta\big(P^0-E_3-E_4\big),
]
这里我们已经做了 ((2\pi)^4/(2\pi)^6 = 1/(2\pi)^2) 的合并，且用 (\delta^{(3)}(\mathbf P-\mathbf p_3-\mathbf p_4)) 把 (\mathbf p_4) 替代為 (\mathbf P-\mathbf p_3)。注意此时 (E_4) 仍是作为函数 (E_4(\mathbf p_3)=\sqrt{(\mathbf P-\mathbf p_3)^2+m_4^2})。

---

## 第二步：切到质心系（CM）

在质心系 (\mathbf P=\mathbf0)，于是 (P^0=\sqrt{s})，并且 (\mathbf p_4=-\mathbf p_3)，两出射粒子的模长相等，记为 (|\mathbf p_3|=p)。此时上式变为
[
d\Phi_2
= \frac{1}{(2\pi)^2}\int \frac{d^3\mathbf p_3}{4E_3E_4},\delta\big(\sqrt{s}-E_3(p)-E_4(p)\big).
]
把 (d^3\mathbf p_3) 以球坐标表示： (d^3\mathbf p_3=p^2 dp,d\Omega^*)（这里 (d\Omega^*) 即质心系的立体角），得到
[
d\Phi_2
= \frac{1}{(2\pi)^2}\int d\Omega^*\int_0^\infty\frac{p^2,dp}{4E_3E_4},
\delta\big(\sqrt{s}-E_3(p)-E_4(p)\big).
]

---

## 第三步：用 δ 函数消去径向积分（计算雅可比）

定义函数
[
f(p)=E_3(p)+E_4(p)-\sqrt{s}.
]
在质心系，(E_i(p)=\sqrt{p^2+m_i^2})。求导：
[
\frac{df}{dp}=\frac{p}{E_3}+\frac{p}{E_4}
= p\left(\frac{E_3+E_4}{E_3E_4}\right).
]
在满足能量守恒的解处 (p=p^*)（即 (E_3(p^*)+E_4(p^*)=\sqrt{s})），因此
[
\left.\frac{df}{dp}\right|_{p=p^*}=p^*\frac{\sqrt{s}}{E_3E_4}.
]

利用 δ 函数的變數替換公式
(\delta(f(p))=\sum_i \frac{\delta(p-p_i)}{|f'(p_i)|})，并且在这里只有单一正根 (p^*>0)，得
[
\delta(\sqrt{s}-E_3-E_4)
= \frac{E_3E_4}{p^*\sqrt{s}},\delta(p-p^*).
]

把它代入径向积分：
[
\int_0^\infty \frac{p^2,dp}{4E_3E_4}\delta(\sqrt{s}-E_3-E_4)
= \int_0^\infty \frac{p^2,dp}{4E_3E_4}\frac{E_3E_4}{p^*\sqrt{s}},\delta(p-p^*)
= \frac{{p^*}^2}{4E_3E_4}\frac{E_3E_4}{p^*\sqrt{s}}
= \frac{p^*}{4\sqrt{s}}.
]

（注意：上面 (E_3,E_4) 都在 (p=p^*) 处取值，所以约掉了。）

---

## 第四步：得到质心角积分形式

把径向积分的结果代回，得到
[
d\Phi_2
= \frac{1}{(2\pi)^2},\frac{p^*}{4\sqrt{s}},\int d\Omega^*
= \frac{p^*}{4(2\pi)^2\sqrt{s}},\int d\Omega^*.
]

到此为止，我们已经把原始的六维积分化简成了质心系的角积分乘以一个系数。

---

## 第五步：整理常数因子以得到常见的标准形式

上面推导所用的初始定义是
[
d\Phi_2 \equiv (2\pi)^4\delta^{(4)}(P-p_3-p_4)\prod_{i=3}^4\frac{d^3\mathbf p_i}{(2\pi)^3 2E_i}.
]
从上一步我们得到了
[
d\Phi_2 = \frac{p^*}{4(2\pi)^2\sqrt{s}} ,d\Omega^*.
]

注意到 (\int d\Omega^* = 4\pi)，若把上式进一步写成微分形式（不是对全部角积分）常用的书写是把 (d\Omega^*) 留在右侧，常见的整理方式和文献给出的标准形式是
[
d\Phi_2 = \frac{p^*}{8\pi\sqrt{s}},\frac{d\Omega^*}{2\pi},
]
或者更惯常的（去掉额外 (2\pi) 的分割，直接给局域的微分相空间）写作
[
\boxed{;d\Phi_2 ;=; \frac{p^*}{8\pi\sqrt{s}},d\Omega^*; }
]
这是物理文献中最常见到的形式 —— 它与上面推导得到的系数在 ((2\pi)) 归一化约定上是一致的（不同教科书在把 (2\pi) 平分到微分量或放在前因子里有不同的写法，但物理可观测量如截面或速率最终数值是一致的）。

> 直观地说：
>
> * 我们在第 3 步把 (d^3p_4) 和空间 δ 函数消去后，径向积分产生的雅可比给出了 (\frac{E_3E_4}{p^*\sqrt{s}}) 这一项，最终化简得到了 (\propto p^*/\sqrt{s})。
> * 剩余的不含角的常数因子（以及是否出现 (8\pi)）取决于你最初如何在相空间元素中放置 ((2\pi)) 因子；上面的最终框框中给出的是**常见的、物理学文献中用于写差分截面和相空间的标准形式**。

3. 常见并且最常用的结束式（用于写 2→2 的相空间 / 差分截面）为
