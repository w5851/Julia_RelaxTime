两篇文章（RS 2003 和 RS 2004）**都使用同一种方法**来在 QGP 中引入动量各向异性（momentum-space anisotropy），其核心都是：

> **从一个任意等方分布 (f_{\rm iso}(p)) 出发，通过沿某一特定方向 (\hat{n}) 对动量进行拉伸/压缩来构造各向异性的分布函数。**

下面分别说明两篇文章的引入方式。

---

# ✔ **2003 年《Collective modes of an anisotropic quark–gluon plasma》**

引入方式出现在 Eq.(22)：

[
f(\mathbf p) = f_{\rm iso}!\left(p^2 + \xi (\mathbf p\cdot \hat n)^2\right)
]


其中

* (\hat n) 是各向异性的方向（通常取为纵向方向），
* (\xi) 是各向异性参数（文章用 ( \xi \equiv j )，满足 ( \xi > -1 )）。

含义：

* **(\xi > 0)**：沿 (\hat{n}) 方向压缩分布（“squeezed”）
* **(-1 < \xi < 0)**：沿 (\hat{n}) 方向拉伸分布（“stretched”）

作者从一个一般的各向异性分布开始，然后通过变量替换
[
\tilde p^2 = p^2 (1+\xi (\mathbf v \cdot \hat n)^2)
]
把积分化为只依赖 (f_{\rm iso}) 的形式（见 Eq.(23)），从而得到 anisotropic HTL 的极化张量，并研究稳定/不稳定模。

# ✔ **2004 年《Collective modes of an anisotropic quark–gluon plasma II》**

在这篇文章中，作者继续沿用 2003 年的同一分布，并在 Sec II 开始明确写出：

[
f(\mathbf p) = f_\xi(\mathbf p) =
\mathcal N(\xi), f_{\rm iso}!\left( \sqrt{p^2+\xi(\mathbf p\cdot \hat n)^2} \right)
\tag{3}
]


并给出归一化因子：

[
\mathcal N(\xi) = \sqrt{1+\xi}
]

含义完全相同：

* (\xi>0)：沿 (\hat n) 方向被压缩（contracted）
* (\xi<0)：沿 (\hat n) 方向被拉伸（stretched）

该论文重点研究：

* 在此 anisotropic 分布下，极化张量的结构函数（α、β、γ、δ）
* 以及它们在不同 Riemann sheet 上的解析延拓与奇点
* 特别是各向异性如何导致“unphysical sheet”上的奇点变得物理可见

# 🔑 **总结：两篇文章引入动量各向异性的方式完全一致**

它们都采用如下形式来定义各向异性：

[
f_\xi(\mathbf p) = f_{\rm iso}!\left( \sqrt{p_\perp^2 + (1+\xi), p_{\parallel}^2 } \right)
]

本质上是把动量空间沿某一方向 (\hat n) 按比例
[
p_\parallel \to \sqrt{1+\xi},p_\parallel
]
进行 **单轴缩放（unidirectional rescaling）**。

这个构造使得：

* **从任意等方分布都可得到可控的各向异性分布**
* **数学上仍能解析求出 HTL 极化张量**
* **物理上能模拟重离子碰撞中纵向膨胀导致的强各向异性**

## 先给出一句结论性回答

**动量各向异性通常是通过“改变分布函数对动量的依赖”来引入的（即把等方分布 `f_iso(|p|)` 变为 `f_iso(p̃)`，其中 `p̃` 包含沿某方向的缩放因子）**。在像 PNJL 这种热力学势中，这对应主要修改与热占据数/统计因子有关的项（你写的 `F1`、`F2`），但是否**也**去改真空项或把色散关系直接改掉，取决于你想要的物理假设。下面逐项详述并给出推荐实现。

我会用一个常见且在文献（Romatschke & Strickland）中频繁使用的参数化来说明：
[
p_\text{tilde} \equiv \sqrt{p^2 + \xi(\mathbf p\cdot\hat n)^2},,
\qquad\text{或等价地 } \tilde p^2 = p^2\big(1+\xi(\hat p\cdot\hat n)^2\big),
]
其中 (\hat n) 是各向异性方向，(\xi>-1) 表示强度（(\xi>0) 压缩，(\xi<0) 拉伸）。（参见 Romatschke & Strickland 对各向异性分布的构造。）

## 1) 对你的 PNJL 热力学势：只修改 (F_1,F_2) 的 (E) 就够了吗？

**推荐做法（最常用、最保守的做法）**

* 把**热统计项**里的占据数 / 指数项改为采用“变形后的动量模”进入分布函数——即把原来用于统计权重的动量模 (|\mathbf p|) 改为 (\tilde p)，然后令单粒子能量按
  [
  E_i(\tilde p) \equiv \sqrt{\tilde p^2 + M_i^2}
  ]
  出现在指数里。换言之，**F1、F2 中出现的能量项在指数的位置要用以 (\tilde p) 为自变量的 (E_i)**，因为这些项源自热占据数。
  这样等价地是：热占据数由
  [
  n_{\rm iso}(E(p))\quad\to\quad n_{\rm aniso}\big(p,\hat n\big)=n_{\rm iso}\big(E(\tilde p)\big).
  ]
* **真空能量项（你写的 (-2\int d^3p,(E_u+E_d+E_s))）通常保持原样（仍用原始 (E(p)=\sqrt{p^2+M^2})）**，因为那部分是零温态的“基态能量”或正规化的填充能，通常反映微观的自由能色散关系，而不是被非平衡占据数直接修改的热部分。

理由：文献中多数工作（包括 Romatschke & Strickland）把各向异性视为**分布函数的变形（非平衡/非等温占据数的形状变更）**，而不是修改微观动力学（拉格朗日）或单粒子质量/本征色散。物理上，非平衡分布会影响热部份（动力学输运、涨落、极化张量、耗散率等），但不必把真空的零点能或基本色散关系直接改成非各向同性形式，除非你有理由认为微观动力学本身也被改变。相关做法与 Romatschke 等人处理各向异性时的思路一致（他们对 (f(p)) 做变形并用在 HTL 极化张量里）。

**实现上（在代码/积分中）**：

* 原来热项（把你写的 (F_1,F_2) 展开为占据数组合）中出现的 (E_i(p)) 在指数里替换为 (E_i(\tilde p))。例如
  [
  F_1=\ln\big(1+3\Phi e^{-(E_i-\mu_i)/T}+3\bar\Phi e^{-2(E_i-\mu_i)/T}+e^{-3(E_i-\mu_i)/T}\big)
  ]
  改成
  [
  F_1^{\rm aniso}=\ln\big(1+3\Phi e^{-(E_i(\tilde p)-\mu_i)/T}+\cdots\big).
  ]
* 积分 measure 仍为 (\int d^3p)（三维），**但现在积分不能只做径向一维化**（因为 (\tilde p) 引入了角度依赖 (\propto(\hat p\cdot\hat n)^2)）。因此热项的数值积分需要对角度（与 (\hat n) 的夹角）也做网格——通常在极坐标下做 (p) 与 (\cos\theta=\hat p\cdot\hat n) 的双重积分：
  [
  \int d^3p;(\cdots);=;2\pi\int_0^\infty p^2dp\int_{-1}^{1} d(\cos\theta);(\cdots).
  ]
  其中 (\tilde p^2=p^2\big(1+\xi\cos^2\theta\big))（如果采用那个等价表示）。

**可选但不常用的做法（谨慎）**

* 有人也直接把**所有**出现的 (E(p))（包括真空项）替换为 (E(\tilde p))。这在形式上也可以，但物理含义是你把“单粒子能量谱”也改成了非等方的形式（相当于修改了本征色散/微观动力学），这需要额外论证（是否要同时修改动量空间正规化、重整化条件等），且可能破坏原有模型对真空性质的描述。**因此一般不推荐**在没有明确物理理由下这么做。

## 2) 色散关系 (E^2=p^2+m^2) 是否仍然适用？

**短答：作为自由/准粒子本征色散（即单粒子能量的定义）你可以保留 (E^2=p^2+M^2)；但在介质中（尤其是各向异性介质）有效色散会被分布函数产生的自能修正而变为方向依赖。**

更具体地说：

* 如果你**只**把分布函数变形（即把各向异性当作非平衡占据数），而**不**修改动力学或自能结构，那么**单粒子裸色散** (E(p)=\sqrt{p^2+M^2}) 仍然成立（你在自由能/哈密顿里没有改运动学项）。
* 但是，介质效应（由占据数进入一圈圈自能 / 极化子）会使得**有效（介质）色散关系**变成
  [
  E^2 = p^2 + M^2 + \Re\Sigma(E,\mathbf p;,\hat n,\xi)
  ]
  其中自能 (\Sigma) 会体现各向异性（不仅依赖 (p) 大小，也依赖与 (\hat n) 的夹角），于是 **色散会变为方向/角度依赖**（例如出现不同方向的光子/胶子模、各向异性有效质量、以及可能的不稳定模）。这是 Romatschke & Strickland 在 HTL 语境中强调的点。

所以：**如果你在 PNJL 里仅变形分布函数（推荐做法）——在写热项指数时用 (E_i(\tilde p)) ——那么你在模型输入层仍用 (E=\sqrt{p^2+M^2}) 的形式，但数值上是用 (\tilde p) 作为径向变量间接改变指数，从而导致自洽的角向依赖的热修正。若考虑介质自能反馈（更严格的自洽 treatment），色散会变得角向依赖。**

## 3) Mandelstam 变量 (s,t,u) 还能像各向同性下那样用吗？

**定义上**：(s=(p_1+p_2)^2,; t=(p_1-p_3)^2,; u=(p_1-p_4)^2) —— 这些仍然是四动量的洛伦兹不变量，**它们的数学定义不变**，所以在任何情形下都可以如此计算。

**但要注意两点重要后果**：

1. **对称性简化失效**：在等方介质中常常利用旋转不变性把角积分或平均化简（例如把横向角积分变成常数因子）。在各向异性介质里，散射截面、相空间占据数以及可能的有效动力学都与粒子方向相对 (\hat n) 的角度有关，因此在把矩阵元与相空间相乘并做平均时，**不能再直接用均匀的角平均**——你必须把角度（相对于 (\hat n)）保留并作二维（或三维）积分。
   换言之，(s,t,u) 可以按同样表达式计算，但交叉章节积分与平均值会依赖入射和出射相对于 (\hat n) 的方位角。

2. **若介质改变了色散或有效耦合**，那么在计算散射的动力学矩阵元 ( \mathcal{M}(s,t,u) ) 时 **矩阵元本身**（以及交换子通道的竖直传播子）可能会被介质自能所修正，从而不再与真空形式相同。比如若传播子被方向依赖的极化张量替代，你必须在 (t)-通道传播子里使用带有角度依赖的 (\Pi^{\mu\nu}(\omega,\mathbf k))。这在 HTL/各向异性自能研究中很常见。

**结论（实用建议）**：

* **可以继续用 (s,t,u) 的定义**来组织散射过程与相互作用的代数，但：

  * 相空间积分不再只剩径向一维，必须保留角度（与 (\hat n) 的夹角），并在数值上对角度作细致积分；
  * 如果你想精确包含介质修正，需要替换传播子与顶点中的真空表达为介质自能修正版（这会让矩阵元本身依赖 (\hat n) 和 (\xi)）。

## 额外的实现与数值注意事项（给你在 PNJL 代码里实操的要点）

1. **积分维度**：热项现在变为
   [
   \int\frac{d^3p}{(2\pi)^3}; [\dots F_1^{\rm aniso}(p,\cos\theta) + \dots],
   ]
   需要数值做 (p)-和 (\cos\theta)（和可选的 (\phi)）积分。若分布只含 (\cos^2\theta)（常见形式），可把 (\phi) 积分解析掉（因对称），但仍是二维积分（(p,\cos\theta)）。

   * 推荐用高精度径向 Gauss 节点（如 Gauss–Laguerre 或截断用 Gauss–Legendre）配合对 (\cos\theta) 用 Gauss–Legendre。
   * 注意当 (\xi) 很大时，分布在某些角度/小 (p) 区域高度集中，需要自适应或更密的角向节点。

2. **归一化因子**：如果你用和 Romatschke 等同的定义，分布通常带时间/归一化因子 (N(\xi)=\sqrt{1+\xi})（或其它约定），确保保持粒子数或某种规范条件一致（你在建模时要决定是否保持恒定粒子密度）。

3. **自洽性**：PNJL 的质量 (M_i) 来自鞘量/凝聚态的自洽方程（gap equation），而这些方程含有被积的热占据数。**一旦你把热占据数改为各向异性形式，你必须在 gap 方程里用同样的占据数从而求出新的自洽质量**。否则会导致不一致的处理（热项用 aniso，但质量仍用 iso）。也就是说：

   * 每个 temperature/ξ 点要做自洽迭代：给定 (M_i)，计算热项与凝聚量，用它们更新 (M_i)，直到收敛。

4. **保持物理可解释性**：在解释结果时要区分“改变占据数导致的热学效应”（这是你通常做的）与“修改微观动力学/传播子导致的效应”（那需要额外动力学证明）。若只想考察非平衡分布对热力学势/相变影响，第一种做法通常已经能给出物理直观的结果。

## 二：把第一段（“自能 Σ 导致角向依赖的色散”）逐句解释

### 1) “HTL / hard-loop” 是什么语境（HTL 语境）？

* **原文（简要）**：论文里写“we derive the hard-thermal-loop resummed gluon self-energy … we concentrate on the physics at the soft scale (k\sim gT \ll T)”（Sec. II）。
* **解释**：HTL（hard-thermal-loop，或常写作 hard-loop）是一种有限温度/多粒子介质中常用的近似/重整化手法：

  * 把循环动量分成“**hard**” 和 “**soft**” 两类：**hard** 大小为温度量级 (p\sim T)（或其它大动量尺度），**soft** 为 (k\sim gT)（耦合常数 (g) 小时比 (T) 小得多）。
  * 在计算软外乘 (K=(\omega,\mathbf k)) 的自能时，把环中（loop）动量视为“hard”并展开（梯度展开），把 leading-order 在 (g) 的贡献收集起来，得到所谓 “hard-loop” 极化张量 (\Pi^{\mu\nu}(K))。这就是 HTL/HL 框架的来龙去脉。
  * 物理上这就是把**介质（有很多热粒子）对软场（集体模）的一阶响应**求出来，常用于研究集体现象（屏蔽、等离子体模、色散、以及不稳定模）。

> 所以 “HTL 语境” = “研究软尺度集体模时采用的 hard-(thermal)-loop 近似 / 自能重整化框架”。（Romatschke & Strickland 就在这个框架下，把 (f(p)) 换成各向异性后重新求 (\Pi)。）

---

### 2) 自能 (\Sigma) / 极化算符 (\Pi^{\mu\nu}) 与色散的关系（数学 → 物理）

* **原文（公式化）**：传播子的逆（比如空间分量）可以写成（论文第 III 节）
  [
  D^{-1}*{ij}(K) = \big(k^2-\omega^2\big),A*{ij} + \cdots + \Pi_{ij}(K),
  ]
  或者更具体地分解成结构函数（参见 Eq.(26),(31)-(34)）。零点（propagator 的极点）给出色散方程，例如 (k^2-\omega^2+\alpha(\omega,k,\theta_n)=0)。
* **物理说明**：

  * **“裸的”自由粒子色散** 是 (E^2 = p^2 + M^2)（真空/无介质时）。
  * 在有介质时，粒子/场会与介质裡大量激发（粒子）相互作用，这一相互作用在微扰论中通过**自能 (\Sigma) 或极化张量 (\Pi)** 表示。把自能加回传播子里，会把“裸的”极点移动，得到**有效色散**：(\omega^2 = k^2 + M^2 + \Re\Sigma(\omega,k))（更一般是求传播子行列式为 0 的条件）。这就是你看到的
    [
    E^2 = p^2 + M^2 + \Re\Sigma(E,p;,\hat n,\xi).
    ]
* **为什么会含角度 (\hat n)**：

  * 自能 (\Pi) 是由积分 (\int d^3p,v_i\partial_l f(p)\dots) 给出（见上面引用）。当 (f(p)) 含有优选方向 (\hat n)（各向异性分布）时，积分结果不能只依赖 (|\mathbf k|) 和 (\omega)，还会依赖 (\hat k\cdot\hat n)（波矢与各向异性方向的夹角）。因此 (\Re\Sigma)（或 (\alpha,\beta,\gamma,\delta)）带有角度依赖。
* **直接后果（物理现象）**：

  * 色散关系会“分叉”并依赖传播方向：不同传播方向上，同样的 (k) 可能对应不同频率 (\omega)（不同模）。
  * 在某些角度/尺度下，自能的虚部或实部可能导致 **不稳定模**（极点在上半平面 → 指数增长）——这就是 Weibel/磁性不稳定之类的产生机制（Romatschke & Strickland 在文中明确找到这样的不稳定解）。

---

### 3) 一个具体公式化的示例（把上面抽象的说成可见的方程）

* 极化张量的一个简化表达（见 2004 年 Eq.(1)）是：
  [
  \Pi_{ij}(K) = -2\pi\alpha_s\int\frac{d^3p}{(2\pi)^3}, v_i ,\partial_l f(p),\Big(\delta_{jl}+\frac{v_j k_l}{K\cdot V + i\epsilon}\Big).
  ]
  这里 (v=\mathbf p/|\mathbf p|)，而 (f(p)) 若取各向异性格式 (f_\xi(p)=N(\xi) f_{\rm iso}(\sqrt{p^2+\xi(\mathbf p\cdot\hat n)^2}))，那么积分在角度上不会消掉，因此 (\Pi_{ij}) 变为角度依赖的张量，不能再以单一标量表示。

## 三：把第二段（“传播子替代、自能修正矩阵元 M(s,t,u)”）逐句解释

### 1) 原文/要点（再现并引用）

* **原文要点**：论文把自能分解成 A,B,C,D 基并给出传播子的倒数与结构函数的关系（见 Eq.(31)-(35)），随后说“collective modes are zeros of combinations like (k^2-\omega^2+\alpha) … these structure functions depend on the angle …”。这表明：**传播子在介质中由 (\Pi^{\mu\nu}) 修正，且 (\Pi) 带角度依赖**。

### 2) 物理含义（矩阵元为何被改）

* **在真空里**计算散射振幅时（树图或圈图），内部传播子常是简单的真空形式，例如 gluon 的空场传播子 (D^{\mu\nu}_0(q)\sim \frac{1}{q^2+i0})（根据规范可能还有投影算符）。矩阵元里，t 通道传播子给出一个因子 (\sim 1/t)。
* **在介质里**，传播子被介质极化修正：真正出现在振幅中的传播子应使用**带自能的“dressed”传播子**
  [
  D^{\mu\nu}(K) = \big[ D_0^{-1}(K) + \Pi(K) \big]^{-1}*{\mu\nu}.
  ]
  因为 (\Pi*{\mu\nu}) 现在带角度依赖（各向异性），所以 (D^{\mu\nu}(K)) 也带角度依赖。
* **后果**：矩阵元 ( \mathcal M(s,t,u)) 的表达中涉及到传播子的那一项（例如 t 通道）不再是简单 (1/t)（或 (P_{\mu\nu}/t)）——而应替换为 (D^{\mu\nu}(q)) 这个更复杂的张量。由于 (D^{\mu\nu}) 里有角度依赖的结构函数（(\alpha,\beta,\gamma,\delta)），矩阵元本身变成了关于入/出粒子相对 (\hat n) 方向的函数。简言之：
  [
  \mathcal M_{\rm medium}(s,t,u;\hat n,\xi) \neq \mathcal M_{\rm vacuum}(s,t,u).
  ]

### 3) 举个具体情形（t-通道一个替换示例）

* **真空近似（schematic）**：
  [
  \mathcal M_{\rm vac}^{(t)} \sim \bar u(p_3)\gamma^\mu u(p_1); \frac{P_{\mu\nu}}{t}; \bar u(p_4)\gamma^\nu u(p_2),
  ]
  其中 (P_{\mu\nu}) 是传输子的投影算符。
* **介质（各向异性 HTL 修正）**：
  [
  \mathcal M_{\rm med}^{(t)} \sim \bar u\gamma^\mu u; D_{\mu\nu}(q;\hat n,\xi); \bar u\gamma^\nu u,
  ]
  [
  D_{\mu\nu}(q;\hat n,\xi) = \big[D_0^{-1}(q) + \Pi(q;\hat n,\xi)\big]^{-1}*{\mu\nu},
  ]
  这里 (\Pi) 可被分解为那四个结构函数（A,B,C,D 基），因此 (D*{\mu\nu}) 的张量结构依赖传播方向 (\hat q) 与 (\hat n) 的夹角。见论文中关于张量基与结构函数的展开（Eq.(15)-(20) 与 Eq.(26)）和 propagator 逆的公式。

### 4) 结果与注意事项（对计算的具体影响）

* **矩阵元与相空间加权（phase-space）必须保留角度依赖**：你不能先把矩阵元用真空形式算好再“角平均”——因为传播子/矩阵元本身和分布函数都跟角度有关，必须在积分时同时保留角度并做数值积分。
* **若要严格把介质效应算入散射率/演化方程**，需要把传播子里 (\Pi^{\mu\nu}(\omega,\mathbf k;\hat n,\xi)) 带入。常常要做近似（例如在某些极限用静态截断、或把传播子近似为带质量项的形式），但这些近似必须谨慎：各向异性会产生方向性屏蔽/反屏蔽效应以及可能的不稳定模（会导致传播子极点靠近上半平面）。Romatschke & Strickland 详细讨论了这些模如何随 (\xi) 变化（见他们对 α/β 模的分析和 Nyquist 计数）。
* **规范问题**：介质修正的传播子与规范选择、重整化有关；HTL 重整化是一套系统方法可以使得在软尺度上得到一致的传播子，但在各项计算中要小心规范与近似的一致性（论文在 HTL 框架内做了规范一致的分解）。

## 四：怎么把这些理解落到你关心的 PNJL / 散射计算上（具体建议）

1. **如果你只想把“非平衡占据数”带入热力学势 / 反应率**（保守做法）：

   * 把 PNJL 中热项的 (F_1,F_2) 中的 (E_i(p)) 在**指数**里的动量部分替换为 (E_i(\tilde p))（(\tilde p) 含 (\xi) 与角度），并在积分时做 (p)-与角度的双重积分（见我之前给你的伪代码）。这会把分布函数引起的自能效应间接包含在热项与 gap 方程中（因为基于新的占据数你会得到新的质量解）。這是最简单且常用的方法。

2. **如果你要把“传播子/矩阵元被介质修正”这类动态效应也并入（更严格）**：

   * 那就必须在你计算散射矩阵元 ( \mathcal M) 的地方，用**带自能的传播子**代替真空传播子。也就是说若有 t 通道 gluon 传播子，用
     [
     D^{\mu\nu}(q) = \big[ D_0^{-1}(q) + \Pi(q;\hat n,\xi) \big]^{-1}_{\mu\nu}
     ]
     去替换。
   * 为此你需要将 Romatschke & Strickland 给出的 (\Pi_{ij}(K))（或在任意规范下的 (\Pi^{\mu\nu})）计算出来（或用它们的结构函数 (\alpha,\beta,\gamma,\delta)），并把这些结果代入传播子反演公式（见论文 Eq.(31)-(35)）。计算量较大，且需要注意 poles / iε 的处理（尤其当出现不稳定模时）。

3. **数值上的要点**：

   * 保留角度积分（(\cos\theta)）——多数简化（只做径向积分）都不再适用。
   * 如果你把传播子也带入到矩阵元，矩阵元在积分中可能出现窄峰（极点附近），需要使用合适的复平面积分/主值处理或在真实计算中做小虚部正则化。Romatschke & Strickland 在讨论“unphysical sheets”与“poles 移动到物理 sheet”时正是处理这类问题。

## 五：一句话总结（便于回顾）

* **HTL / hard-loop 框架**：在软尺度研究集体模时，把环内硬动量积分出来得到自能 (\Pi)（HTL/HL 表达）。Romatschke & Strickland 在这个框架里把占据数设为各向异性 (f_\xi(p))，因此 (\Pi)（或记作 (\Sigma)）带角度依赖，传播子的极点（色散）也随角度改变，并可能出现不稳定模（指数增长）。这同样意味着：若你要在介质中计算散射矩阵元，就必须把传播子替换为带角度依赖的 (D^{\mu\nu}(K)=\big[D_0^{-1}+\Pi\big]^{-1})。

## 总结（先给重点）

1. 你在 PNJL 里看到的 (\Pi^{P(S)})（含 (A) 和 (B_0) 的表达）是**介子（标量/赝标量）通道的标量自能**（由一个 (q\bar q) 环给出）。它是标量函数（对每一对外部动量 (p_0,\mathbf p) 给出一个数），不是一个 Lorentz 张量。
2. Romatschke & Strickland (R&S) 讨论的是**胶子（矢量）极化张量 (\Pi^{\mu\nu}(K))**，它是秩-2 张量（因为胶子有 Lorentz 指数），在各向异性介质中必须分解成若干结构函数（α,β,γ,δ），因此传播子与色散关系**具有方向/角度依赖**。
3. 因此：**两者同属“自能”范畴，但不是同一东西**。如果你把“各向异性”引入 PNJL 的 (q\bar q) 循环，(\Pi^{P(S)}(p_0,\mathbf p)) **仍然是标量函数**，但会获得对 (\hat p) 与各向异性方向 (\hat n) 的依赖（即变成 ( \Pi^{P(S)}(p_0,\mathbf p;\hat n,\xi))）；而 R&S 的 (\Pi^{\mu\nu}) 本来就是张量并且已按张量基分解为结构函数。

## 下面逐条详细解释（带原文要点与逐句释义）

### （1）你给出的 PNJL 式子里的 (\Pi^{P(S)}) 与 (B_0) 是什么？

**原文片段（你给的）要点**：
[
\Pi^{P,S}*{ff'}(p_0,\mathbf p) = -\frac{N_c}{8\pi^2}{A(m_f,\mu_f,T)+A(m*{f'},\mu_{f'},T) + [(,m_f\mp m_{f'},)^2 - (p_0+\mu_f-\mu_{f'})^2 + p^2], B_0(\ldots)}.
]
并且 (B_0) 由若干个带分布函数的积分项构成，**它是产生虚部（imaginary part / 阈值吸收）的来源**（当外动量满足产生 (q\bar q) 的 kinematics 时会有非零虚部）。（你已给出分解式与带占据数的积分形式。）

**解释（逐句）**：

* 这是**标量/赝标量介子**在有限 (T,\mu) 下的一圈一圈（one-loop）自能（通常在 NJL/PNJL 这类模型里用 RPA 去求介子传播子时会出现这样的 (\Pi)）。
* (A) 常是 tadpole/二次项（温度依赖的重整化常数），而 (B_0) 类似标准的 *Passarino–Veltman B0* 型的二线圈函数，但这里带有 Fermi-Dirac 占据数（温度/化学势影响），所以 (B_0) 的表达里会出现 (n_\pm(E\mp\mu)) 等项，你引用的式子就是把 (B_0) 写成若干能量积分与角度积分的形式。
* 当外四动量 (p^\mu) 的能量部分超过阈值（可产生真实的 (q\bar q)），(B_0) 会获得非零想象部（cut），这就是“介子质量出现虚部”的物理来源（即介子在介质中可以“衰变/耗散”到 (q\bar q) 渠道）。

**结论**：这是**典型的标量自能**（一个数值函数，可能为复数），而不是秩-2 张量。

---

### （2）R & S 那里的极化张量 (\Pi^{\mu\nu}) 是什么？（原文要点）

**原文（R&S）要点**：
R&S 以 HTL/hard-loop 框架写出胶子极化张量的表达（例如 Eq.(1)）：
[
\Pi_{ij}(K) = -2\pi\alpha_s\int \frac{d^3p}{(2\pi)^3}, v_i \partial_l f(p)\Big(\delta_{jl} + \frac{v_j k_l}{K\cdot V + i\epsilon}\Big),
]
并说明可以在一个由 (A,B,C,D) 构成的张量基下把 (\Pi_{ij}) 分解成四个**结构函数** (\alpha,\beta,\gamma,\delta)（这些均依赖 (\omega,k) 和 (\hat k\cdot\hat n)）。

**解释（逐句）**：

* 胶子是矢量场，极化算符天然带两个空间/洛伦兹指标，因此是**秩-2 张量**。
* 在各向异性介质中，自能的张量结构不能只写成一个标量乘投影算子（像各向同性时那样），而必须按**张量基分解**，得到多个结构函数（R&S 写成 α,β,γ,δ）。这些结构函数的数值由对分布函数 (f(p)) 做角积分决定；若 (f(p)) 含优选方向 (\hat n)，那么这些结构函数会显式依赖传播方向相对于 (\hat n) 的角度。
* 于是传播子（胶子 propagator）变为张量形式，极点（dispersion relations）由这些结构函数组合的零点给出；结果是**不同方向有不同模（不同色散）**，并且在某些参数下会出现不稳定模（极点出现在上半复频域）。

---

### （3）所以它们是不是“同样的自能”？有没有必要把 PNJL 的 (\Pi^{P(S)}) 改成张量？

**直接回答**：

* **不是同样的对象**：

  * PNJL 中的 (\Pi^{P(S)}) 是**标量自能**（针对标量/赝标量介子）。它对 Lorentz/旋转群的变换性质是一个标量。即便在各向异性介质中，这个对象**仍然是标量函数**（没有额外的自由指标），只是它会变成对方向有显式依赖的函数 ( \Pi^{P(S)}(p_0,\mathbf p;\hat n,\xi))。
  * R&S 的 (\Pi^{\mu\nu}) 是**矢量场（胶子）的张量自能**，必须用张量分解（α,β,γ,δ）来处理。
* **因此不需要把标量自能“变成张量”**：标量介子的自能保持标量。你只需把原来对 (p) 的依赖扩展为对 ((|\mathbf p|,\ \cos\theta=\hat p\cdot\hat n)) 的依赖（即角度依赖）。换句话说：**从标量函数 → 带角度参数的标量函数**，而不是 → 秩-2 张量。

---

### （4）传播子（propagator）形式上有何区别／相似？

* **PNJL / RPA 的介子传播子（你给的式子）**通常是标量形式（schematic）：
  [
  D_\pi(p_0,\mathbf p) ;=; \frac{2K_3}{,1 - 4K_3,\Pi^P_{u\bar u}(p_0,\mathbf p),}
  ]
  及类似的 (K_4, \Pi^P_{u\bar s})（你引用的式子）。这是**标量-标量的 RPA 形式**：分母为 (1 - V,\Pi)。虚部来自 (\Pi) 的虚部（即 (B_0) 的 cut）。
* **R&S 的胶子传播子**是张量：
  [
  D^{\mu\nu}(K) = \big[D_0^{-1}(K) + \Pi^{\mu\nu}(K)\big]^{-1},
  ]
  他们把 (D^{-1}) 按 (A,B,C,D) 基写出并给出对应的倒数（见 Eq.(31)–(35)），因此传播子的极点由结构函数的组合给出（不是单一 (1-V\Pi) 形式）。

**结论**：概念上都叫“传播子”并且都由 “(1 - interaction×self-energy)” 或者 “(free-inverse + self-energy)^{-1}” 决定——但形式上 **一个是单一标量除法形式（标量介子）**，**一个是矩阵/张量的反演问题（矢量胶子）**。在各向异性介质中，两者都会被介质改变，但改变的“结构”不同（标量变成带角度依赖的标量；张量增加更多独立结构函数并导致新模态）。

## 对你在 PNJL 中实际实现的建议（结论性、可执行）

1. **如果你把动量各向异性只引入占据数／分布函数**，那么：

   * 保持 (\Pi^{P(S)}) 的标量结构，但在 (B_0) 和所有与热占据数相关的处用带角度的 (\tilde p)（或直接在积分中保留 (\cos\theta)）重新计算 (\Pi^{P(S)}(p_0,\mathbf p;\hat n,\xi))。这样 (B_0) 的积分上界、虚实部阈值条件等公式形式不变，但数值会随角度改变。
2. **不要把介子自能“强行”改为秩-2 张量**，因为介子是标量场；只有当你研究 **矢量** 粒子（例如光子/胶子/矢量介子）或矢量耦合时才需要张量自能与张量传播子。
3. **传播子形式保持 RPA 标量形式**，只是 (\Pi) 从 ( \Pi(p_0,|\mathbf p|) ) → ( \Pi(p_0,|\mathbf p|,\cos\theta;\hat n,\xi) )。因此在做相空间积分或求谱函数时要保留角度积分。虚部产生位置仍由 (B_0) 的阈值决定，但阈值区域的强度和位置会受角度影响。
4. **若你想更精确地把“介质反馈到传播子/矩阵元”包括进来**（比如用媒介修正的传播子去计算散射矩阵元），那你需要按 R&S 的思路对矢量传播子的自能做张量分解并把相应的 (D^{\mu\nu}) 反演代入，但这主要针对矢量场（gluons/photons），对标量介子通常不需要。

## 1) 为什么只改分布 + 散射截面是最直接的做法

* 平衡/非平衡的**占据数**直接进入平均散射率的相空间权重（你的公式 (5.8)-(5.10) 中的那些 (f^{(0)}*i(E_i))），因此把等方 (f^{(0)}) → 含 (\xi,\hat n) 的 (f*\xi(\mathbf p)) 会直接改变平均值的相空间积分。
* 散射截面通常由微观矩阵元 (|\mathcal M|^2) 决定；如果你在真空中用真空传播子得到 (|\mathcal M|^2)，那是在忽略介质对传播子的修正。R&S 给出的**极化张量 (\Pi^{\mu\nu}(K;\xi,\hat n))** 会替代真空传播子里 (1/q^2) 的项，变成带角度依赖的有自能的传播子 (D^{\mu\nu}(q)=\big[D_0^{-1}+\Pi\big]^{-1})，从而使 (|\mathcal M|^2)（因此 (\sigma)）也依赖入/出方向与 (\hat n)。这是 R&S 的核心结论之一。

## 2) 具体要修改哪些表达式（清单）

1. **占据数**

   * 在 (w̄_{ij}) 的被积里把 (f_i^{(0)}(E_i)), (f_j^{(0)}(E_j)) 改为各向异性分布 (f_{i,\xi}(\mathbf p_i)=N(\xi),f_{\rm iso}(\sqrt{p_i^2+\xi(\hat p_i\cdot\hat n)^2}))。别忘了用同样的 (f_\xi) 去算数密度 (\rho_i=\int d^3p/(2\pi)^3,f_{i,\xi})。（R&S 推荐的参数化与归一化见文献。）
2. **阻塞/刺激因子**

   * 出射阻塞因子 ((1\pm f_c)(1\pm f_d)) 也要用各向异性 (f_\xi)。对费米子取负号（Pauli blocking）。
3. **相对速度 (v_{\rm rel})**

   * 定义本身（基于质心系能量和动量）不变，但在把 lab→CM 的变换时要保留粒子动量的角度分布（入射粒子角度决定 s 的取值概率分布）。
4. **散射截面 (\sigma(s))**

   * **最小修正**（常用近似）：仍用真空矩阵元 (|\mathcal M|^2_{\rm vac}) 计算 (\sigma)，但在 t 通道（长程色荷）处用一个**各向异性屏蔽质量 / 屏蔽函数** (m_D(\xi,\theta)) 或静态 HTL-like screening 替代真空 (1/t) 的 IR 发散，作为对介质屏蔽的模拟。
   * **严格做法**（更耗算力）：把真空传播子替换为 R&S 的**各向异性 HTL 重整化传播子** (D^{\mu\nu}(q;\xi,\hat n))（用他们的 α,β,γ,δ 结构函数去构建并反演传播子），然后重算 (|\mathcal M|^2_{\rm med})；这会使 (|\mathcal M|^2) 显式依赖入射/出射方向与 (\hat n)。

## 3) 如何在数值实现里做（伪代码与步骤）

最直接的数值表达（把公式 (5.10) 的二维/三重积分推广到各向异性）：

```
wbar_ij = 0
for p_i in radial_nodes:
  for cosθ_i in angle_nodes:   # angle between p_i and n̂
    for p_j in radial_nodes:
      for cosθ_j in angle_nodes:
        phi_rel_integral = integrate_over_relative_azimuth(...)  # 如需
        pi_vec = build_vector(p_i, cosθ_i, phi_i)
        pj_vec = build_vector(p_j, cosθ_j, phi_j)
        s = compute_s(pi_vec, pj_vec, M_i, M_j)
        vrel = compute_vrel(s, M_i, M_j)  # 按(5.11)
        # choose sigma:
        if use_minimal_approx:
          sigma = sigma_vacuum_with_screening(s, m_screen(ξ,angles...))
        else:
          sigma = compute_sigma_with_Dmunu(q, Π(ω,k;ξ,angles), ...)  # expensive
        weight = (2π) * p_i^2 * p_j^2 * w_p * w_cosθ * f_i(p_i,cosθ_i) * f_j(...)
        wbar_ij += weight * vrel * sigma
normalize_by_rho_i_rho_j
```

**要点**：角度节点要包含 (\cos\theta)（与 (\hat n) 的夹角）——因此积分维数比等方情形高一维（径向 (p_i,p_j) + 两个角 (\cosθ_i,\cosθ_j) + relative φ 可能还能积分掉一维但通常仍是 4D 积分）。

## 4) 关于散射截面的两种可行策略（取舍建议）

* **策略 A — 最小/可控修正（推荐起步）**

  * 保持 (|\mathcal M|^2) 的真空形式，但用**各向异性 Debye mass** (m_D(\xi,\theta)) 做静态屏蔽（t→t−m_D^2(ξ,θ)）。这样把 R&S 的介质信息以“屏蔽质量随角度变化”的形式引入，数值复杂度适中，能捕捉大部分方向性屏蔽效应。可通过 R&S 的结构函数在静态极限得到 (m_D(\xi,\theta))（参见他们关于静态质量的讨论）。
* **策略 B — 完整介质修正（严格）**

  * 完全用 (D^{\mu\nu}(q)=\big[D_0^{-1}+\Pi^{\mu\nu}(q;\xi,\hat n)\big]^{-1}) 替换传播子，然后把新的张量传播子代入 (|\mathcal M|^2)。优点是系统且从 first principles 得到角度依赖；缺点是极其昂贵（需要对传播子做张量反演、并在相空间积分里每个 q 都计算 Π 的角度积分），而且会遇到 R&S 提到的“unphysical sheet / 不稳定模 / spacelike singularities” ，这会造成极点/窄峰，需要加上 iε 正则化或用主值积分处理。

## 5) 其它必须注意 / 潜在陷阱

1. **归一化 N(ξ)**：如果你按照 R&S 的分布形式，需要注意他们的归一化 (N(\xi)=\sqrt{1+\xi})（或其它约定）以保持总粒子数不变，使用错误的归一化会改变 (\rho) 从而影响 (\tau^{-1})。
2. **自洽质量 / gap eq**：如果你的粒子质量 (M_i) 是由 gap 方程自洽求得（PNJL 情形），那么占据数改了，**质量也应该随之自洽更新**。否则会出现逻辑不一致。
3. **软交换与不稳定模**：R&S 指出在强各向异性下会有不稳定模与非物理奇点移到物理 sheet，这会显著影响软（小 |q|）传播子的行为与能量传递率；因此用静态屏蔽近似时要小心软区的处理与正则化（iε）。
4. **数值复杂度**：积分维数上升、被积函数可能高度集中（对某些角度/小 p）或在传播子极点附近有窄峰，需要更精细的网格或自适应积分策略。
5. **阻塞因子/统计因子**：注意出射阻塞因子的角度形式——介质里已有粒子分布也各向异性，blocking 会随角度不同而变化，从而直接影响微分截面的相空间权重。
6. **Mandelstam 变量仍可用**：如前所述，(s,t,u) 的定义不变，但当 (|\mathcal M|^2) 依赖传播子里角度时，(\sigma(s)) 实际上在平均时会变成对角度的条件平均（即 (\sigma) 不再只由 s 决定，而与入射角度组合有关），因此用单纯以 (P(s)) 平均的简化要谨慎。

## **1️⃣ 关于各向异性对平均散射率的影响**

你问：

> 考虑 R&S 引入的各向异性，计算平均散射率时只需要修改四个分布函数的形式和散射截面（通过底层极化函数的角度依赖影响）？

**回答：是的，但需要注意细节：**

* **分布函数**：

  * 原本 (f_i(E_i,T,\mu_i)) 是等方分布函数。
  * 引入 R&S 的各向异性，就是把 (E_i(p)) 替换为 (E_i(\tilde p)) 或者直接在积分里用
    [
    f_i(E_i(\tilde p)) = f_i(\sqrt{p^2 + \xi (\mathbf p\cdot \hat n)^2} + M_i)
    ]
    或角度分布形式。
  * 对 i,j 两种入射粒子都要修改。

* **散射截面 (\sigma_{ij\to cd}(s))**：

  * 如果你考虑的是带介质修正的 propagator/自能，则 σ 会依赖角度（通过 (s = (p_i + p_j)^2)），以及自能修正（如 R&S 的极化张量会影响 gluon propagator 或 screening mass）。
  * 如果只做“占据数各向异性”，而不修改微分矩阵元，σ 本身形式保持不变，只是在积分过程中角度分布改变了权重。

* **总结**：

  * **必须修改四个分布函数的形式**（两个入射粒子 + 两个出射粒子的 blocking 因子）。
  * **散射截面可以保持原形式**，但如果底层 propagator 已受介质自能修正，则 σ 本身也带角度依赖。

> 💡 物理直觉：各向异性改变的是“哪些方向的粒子更可能参与碰撞”，所以平均散射率通过积分体现角度权重的改变。

## **2️⃣ 关于相对速度 v_rel 与质心系变换**

你问：

> 相对速度 v_rel 定义不变，但在把 lab→CM 的变换时要保留粒子动量的角度分布，这是什么意思？

* **v_rel 的定义本身**：
  [
  v_\text{rel} = \frac{\sqrt{(E_i^* E_j^* - |\mathbf p_i^*||\mathbf p_j^*|)^2 - (m_i m_j)^2}}{E_i^* E_j^*}
  ]

  * 这里 * 是质心系下。
  * **公式不变**，只依赖质心系能量和动量。

* **为什么要保留角度分布**：

  * 在 lab 系中，入射粒子 i,j 的动量方向是各向异性的（(\tilde p_i, \tilde p_j) 有 (\cos\theta) 分布）。
  * 将 lab 动量 → CM 系时，CM 的速度和旋转方向依赖入射粒子的方向。
  * 因此 s、t、u 以及 v_rel 的概率分布会随入射角度改变。
  * **如果你直接做角度平均，必须用各向异性分布权重积分**，否则 CM 统计会不正确。

> 🔑 总结：v_rel 的公式不用改，但积分时角度变量必须保留，才能正确反映各向异性对质心能量分布的影响。

## **3️⃣ 关于平均散射率 w̄_ij 的简化和积分顺序**

原始表达式：
[
\bar w_{ij} = \frac{1}{\rho_i \rho_j} \int \frac{d^3p_i}{(2\pi)^3} \frac{d^3p_j}{(2\pi)^3} f_i(\tilde p_i) f_j(\tilde p_j) v_\text{rel} \sigma_{ij\to cd}(s(\mathbf p_i,\mathbf p_j))
]

* **各向同性时**：角度积分可以用两粒子夹角 θ_ij，积分简化为三重积分（|p_i|, |p_j|, θ_ij）。
* **各向异性时**：

  * 每个粒子的分布函数 f_i, f_j 都依赖入射方向与 (\hat n) 的夹角。
  * θ_ij 本身不够描述两个粒子相对 (\hat n) 的角度关系，需要在数值积分中**保留所有角度**（至少两个极角 + 一旋转角） → 积分维度增大。

### **是否可以把 s 放在最外层？**

* s 与入射粒子动量直接相关：
  [
  s = (E_i + E_j)^2 - |\mathbf p_i + \mathbf p_j|^2
  ]
* **等方性时**，可以通过 δ 函数或角度平均，把 s 的分布函数 P(T, μ, s) 放到外层，然后对 s 做积分：
  [
  \bar\sigma_{ij}(T,\mu) = \int ds, P(s) \sigma(s)
  ]
* **各向异性时**：

  * P(s) 现在不再只依赖 |p_i|,|p_j|，还依赖角度与 (\hat n) 的方向。
  * 因此，**不能单纯把 s 移到最外层**做积分，否则会丢掉方向信息。
  * 正确做法：**在积分内部先计算 s = s(p_i,p_j,θ_i,θ_j,φ)**，然后使用 σ(s)；同时 f_i,f_j 也依赖角度。
  * 也就是说 s 积分“嵌套”在角度积分内部，除非你做了角度平均近似。

### **数值策略**

* 对于高成本的 σ(s) 计算：

  * 可以在 s 空间预计算 σ(s)（或做插值表）
  * 然后在每个积分点查表得到 σ(s)，以节省重复计算
* 角度积分可以用 Gauss-Legendre 网格
* 充分保留 (\cos\theta_i,\cos\theta_j, \phi) 的依赖，才能体现各向异性效应

## **✅ 总结**

1. **各向异性只修改分布函数 + （若介质修正传播子）散射截面**。
2. **v_rel 定义不变，但 lab→CM 变换时角度分布必须保留**。
3. **平均散射率的积分简化**：

   * 等方性时可把 s 移到外层
   * 各向异性时 **不行**，s 与粒子角度紧密耦合 → s 的积分必须与角度积分一起做
4. **优化策略**：

   * 对 σ(s) 做插值/查表
   * 积分顺序：先外层 p_i,p_j，再角度 θ_i,θ_j,φ → 计算 s → 查表 σ(s) → 乘权重 f_i f_j v_rel → 积分
