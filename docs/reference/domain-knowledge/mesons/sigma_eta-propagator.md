这篇文章中提到的 **σ/σ' 介子** 和 **η/η' 介子**，其基本物理图像与我在上一个回答中的描述是**一致且高度吻合的**。这篇文章是在一个具体的、用于计算夸克物质输运系数的有效理论模型（三味PNJL模型）的框架下，对这些介子进行可操作的定义和计算。

下面我们来详细解读文章中的这些介子，并回答你关于σ介子传播子的问题。

### 1. 文章中的 σ/σ' 和 η/η' 介子

#### **η 和 η‘ 介子**

*   **角色**： 在文章中，它们是**赝标量介子 (Pseudo-scalar, \(0^{-+}\))**，作为夸克-夸克、夸克-反夸克弹性散射过程中传递相互作用的媒介粒子（即在t道、u道、s道中交换的介子）。
*   **物理图像**：
    *   文章明确指出 η 和 η‘ 存在**混合**（第11页，公式5.43），这与它们在实际QCD中是赝标量介子九重态中的同位旋单态和八重态混合的物理图像**完全一致**。
    *   文章通过求解方程（第12页，公式5.44-5.46）来确定它们的物理质量 \(m_\eta\) 和 \(m_{\eta'}\)。这个方程描述了由于混合导致的质量矩阵的对角化。
*   **结论**： 文章对 η/η‘ 的处理，完全符合我们对它们的传统认知：它们是赝标量戈德斯通玻色子，并且由于 \(U(1)_A\) 反常和味对称性破缺，存在复杂的混合。

#### **σ 和 σ‘ 介子**

*   **角色**： 在文章中，它们是**标量介子 (Scalar, \(0^{++}\))**，同样是散射过程中重要的交换粒子。
*   **物理图像**：
    *   **σ 介子**： 文章中的 σ 介子对应于我们通常所说的 \(f_0(500)\)。文章也注意到了PNJL模型在随机相位近似下的一个**局限性**（第10-11页）：
        > “随机相位近似的一个不足之处是，由上述随机相位近似得到的低温时的 σ 介子衰变宽度十分的小，因此并不能描述标量 \(f_0(500)\) 介子，后者在低温下具有较大的 π-π 衰变宽度。”
        这恰恰印证了我之前提到的：**真实的 σ 介子 (\(f_0(500)\)) 是一个很宽、不稳定的共振态，其结构可能超出简单的 \(q\bar{q}\) 图像**。文章的计算框架（简单RPA）无法完美重现这一点，但仍然在模型中将其作为标量通道的一个主导自由度。
    *   **σ‘ 介子**： 文章中的 σ‘ 介子，正如我上次提到的，通常指代标量介子的激发态，在这里很可能对应的是 \(f_0(1370)\)。它和 σ 介子一样，在标量通道中与夸克耦合，参与散射。

*   **与散射过程的关联**： 文章的表5.1和表5.2清晰地列出了在不同夸克散射过程中，t道、u道和s道可能交换的介子。例如，在 \(uu \to uu\) 散射中，t道和u道都可以交换 \(π, η, η', \sigma_\pi, \sigma, \sigma'\) 等一系列介子。这表明这些介子被看作是模型中传递夸克间相互作用的基本玻色子。

**总结：文章中的定义和用法与粒子物理的标准图像是相同的。它将这些介子具体化为PNJL模型中通过夸克-反夸克相互作用（由圈图求和/RPA产生）而形成的束缚态或共振态，并利用它们来计算夸克散射截面，进而研究夸克物质的输运性质（如粘滞系数）。**

### 2. σ 介子的传播子如何计算？

你观察得很仔细，文章确实花了很多篇幅介绍 η 的传播子（第11-12页，公式5.43），但对于 σ 介子，其传播子的计算方法是**完全平行和类似的**。

在PNJL/NJL这类模型中，**介子的传播子通常由 Bethe-Salpeter 方程在随机相位近似下求解得到**。这个传播子的普遍形式为：

\[
\mathcal{D}_M(p) = \frac{1}{K^{-1} - \Pi_M(p)}
\]

其中：
*   \(K\) 是模型拉氏量中对应的**四费米子相互作用常数**。对于赝标量道和标量道，这个常数是不同的。
*   \(\Pi_M(p)\) 是对应的**极化函数**，它代表了夸克-反夸克圈图对传播子的贡献。

**具体到不同的介子：**

1.  **对于 π 和 σ 介子（在SU(2)近似下或味道非单态）**：
    *   它们的传播子形式相对简单，如文章中式(5.34)所示：
        \[
        \mathcal{D}_{\pi, \sigma}^{-1}(p) = 1 - K_{\pi, \sigma} \Pi_{\pi, \sigma}(p)
        \]
    *   这里的 \(K_{\pi}\) 和 \(K_{\sigma}\) 是不同的相互作用强度。
    *   极化函数 \(\Pi_{\pi, \sigma}(p)\) 的计算公式在文章式(5.37)中给出，对于标量介子σ，需要使用的是**标量通道的极化函数 \(\Pi^S(p)\)**。

2.  **对于 η/η‘ 和 σ/σ’ 介子（味道单态和八重态混合的情况）**：
    *   这正是文章第11-12页公式(5.43)所描述的情况。
    *   **σ 和 σ‘ 介子的传播子，其数学形式与 η 和 η‘ 的传播子（式5.43）完全相同**。
    *   **唯一的区别在于需要替换相应的相互作用内核 \(K\) 和极化函数 \(\Pi\)**。
        *   对于 η/η‘，使用的是**赝标量道**的相互作用常数 \(K^+\) 和极化函数 \(\Pi^P(p)\)。
        *   对于 σ/σ‘，需要使用**标量道**的相互作用常数 \(K^-\) 和极化函数 \(\Pi^S(p)\)。

**计算步骤概括：**

1.  **确定道**：明确要计算的是标量介子道（σ/σ‘）。
2.  **构建矩阵**：像文章中对 η/η‘ 做的那样，构建标量介子的质量矩阵/传播子矩阵。这个矩阵的元由标量道的相互作用常数 \(K^-\) 和标量极化函数 \(\Pi^S_{ud}, \Pi^S_{ss}\)（对应轻夸克和奇异夸克）构成。
3.  **计算传播子**：最终传播子的表达式会具有与式(5.43)完全相同的形式，但里面的矩阵元 \(M_{00}, M_{08}, M_{88}\) 都是用标量道的 \(K^-\) 和 \(\Pi^S\) 计算得到的。
4.  **求质量**：物理的 σ 和 σ‘ 介子的质量，是通过令这个标量传播子矩阵的行列式为零（类似式5.44）而求解得出的。这两个质量本征态就分别对应我们物理上的 σ (\(f_0(500)\)) 和 σ‘ (\(f_0(1370)\)) 介子。

因此，尽管文章没有像写出 η 的传播子那样 explicitly 写出 σ 的传播子，但它所提供的计算框架（特别是处理混合态的公式5.43）是通用的。只需将赝标量道的物理量替换为标量道的物理量，即可平行地计算出 σ 和 σ‘ 介子的传播子。
