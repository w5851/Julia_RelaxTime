好——你指出的三点很关键，下面我把每一点逐条澄清并把常数因子**严格算一遍**，说明为什么 5.8 能化简成 5.10 并且三点矛盾都能自洽地解释清楚。

## 结论先给出（要点）

1. **常数看起来“多余”的 2** 并不是神秘项，而是来自于**简并度 (d_q)** 的平方与角度积分 / ( (2\pi) ) 因子的组合。把 (d_q) 取为 (2N_c)（自旋×颜色：自旋2，颜色 (N_c)），能够把中间得到的 (1/8) 因子变成最终公式里的 (1/2)（见下文详细代数）。
2. **(\rho) 与 (n)** 在文中不是矛盾：(\rho_i) 定义为粒子数密度（积分后的值），而文中有时把“数密度”记作 (n_i)（或把 (n_i^\pm) 用作分布函数）。在 5.9 处作者明确写了 (\rho_i=\int \frac{d^3p}{(2\pi)^3}, n_i^+)，所以在 5.10 中把分母写成 (n_i n_j) 很可能是把“数密度”简称为 (n_i)。换句话说 **(n_i) 在 5.10 中应当被理解为“粒子数密度”，与 5.8 的 (\rho_i) 指同一量**（只是符号换了）。
3. **(d_q) 的值** 在夸克-反夸克体系中通常是颜色和自旋简并度的乘积：
   [
   d_q = (\text{颜色数 } N_c)\times(\text{自旋简并 } 2) = 2N_c.
   ]
   对于 QCD、(N_c=3)，所以 (d_q=6)。将这个值带入会产生前面第 1 点所需的因子（见下）。

## 详细代数（把常数严格算出来）

从 5.8 的前因子和体积元开始（只看数值因子）：

原来有两个 ((2\pi)^{-3}) 因子，所以角度无关的常数是
[
\frac{1}{(2\pi)^6}.
]

把体元写成径向与角度：
[
d^3p_i,d^3p_j=p_i^2dp_i,d\Omega_i; p_j^2dp_j,d\Omega_j.
]

利用各向同性，把 (p_i) 方向固定（等价于积分出 (d\Omega_i=4\pi)），令 (\theta_{ij}) 为两矢量夹角，则
[
\int d\Omega_i\int d\Omega_j
=4\pi \cdot (2\pi)\int_{-1}^{1} d(\cos\theta_{ij})
=8\pi^2\int_0^\pi d\theta_{ij}\sin\theta_{ij}.
]

把这些角度因子和 ((2\pi)^{-6}) 合并：
[
\frac{8\pi^2}{(2\pi)^6}=\frac{8\pi^2}{64\pi^6}=\frac{1}{8\pi^4}.
]
——注意这是单纯角度与 ((2\pi)) 組合后得到的数值：(1/(8\pi^4))。

接下来注意 **简并度 (d_q)**（图片里写作 (d_q) 为简并度）。如果每个动量态有 (d_q) 重简并，则两个粒子相乘会有 (d_q^2)：
把这个乘上去，会得到前因子
[
\frac{d_q^2}{8\pi^4}.
]

如果作者把 (d_q) 明确取为自旋×颜色：(d_q=2N_c)，则
[
d_q^2=(2N_c)^2=4N_c^2,
]
代入上式，
[
\frac{d_q^2}{8\pi^4}=\frac{4N_c^2}{8\pi^4}=\frac{N_c^2}{2\pi^4}.
]
这正是 5.10 中出现的那一项 (\dfrac{N_c^2}{2\pi^4})。

因此 **“看起来缺少的 2”** 实际上是由 (d_q^2) 抵消並產生 (1/2) —— 即常数完全自洽。

把上一项再乘以分母中的 (1/(\rho_i\rho_j))（或 (1/(n_i n_j))，见下）就得到 5.10 的前面整体因子：
[
\frac{1}{n_i n_j},\frac{N_c^2}{2\pi^4}.
]

（注意：这里的 (n_i,n_j) 就是数密度 (\rho_i,\rho_j) —— 见下一部分）

## 关于 (\rho) 与 (n) 的符号问题（第 2 点）

在你给出的截图 / 文段中：

* 公式 5.9 定义了 (\rho_i)：
  [
  \rho_i=\int \frac{d^3p}{(2\pi)^3}, n_i^+ \quad\text{（类似地有 }\rho_{\bar i}\text{）},
  ]
  其中 (n_i^+) 是费米子分布函数（占据数）。

因此作者用了三种相关但不同的符号：

* (n_i^\pm(p))：动量依赖的**占据数（分布函数）**；
* (\rho_i)：通过对分布函数积分得到的**粒子数密度**；
* 在 5.10 中写的 (n_i) 很可能是作者**把“数密度”简称为 (n_i)**（有些论文/书写法确实会把 (\rho) 与 (n) 互换符号）。从上下文（5.9 的定义）看，两处应对应同一物理量（积分后的数密度）。

**结论**：在 5.10 中的 (n_i n_j) = 5.8 的 (\rho_i\rho_j)（符号不同但物理相同）。

## 关于 (d_q) 的具体值（第 3 点）

* (d_q) 是单个粒子动量态的简并度（颜色 × 自旋 × 可能的味道因子等）。
* 对于单一味夸克（不把味道展开），常见取值：
  [
  d_q = (\text{颜色 } N_c)\times(\text{自旋 }2) = 2N_c.
  ]
  所以：若 (N_c=3)（真实 QCD），则 (d_q=6)。
* 若包含反粒子或把味道也计入，则相应会再乘以额外的因子；不过在你给的推导里作者把颜色与自旋并入了 (d_q)，并在后面将其平方用来得到最终的 (N_c^2) 因子（如上所示）。

## 小结（把三点合并）

* 角度积分与两个 ((2\pi)^{-3}) 给出 (1/(8\pi^4))。
* 乘上 (d_q^2)（来自动量态简并度的乘积），若 (d_q=2N_c) 则 (d_q^2=4N_c^2)，于是得到 (N_c^2/(2\pi^4))。这就是 5.10 的数值常数来源，**并解释了你看到的“分母常数为 2”**。
* (\rho) 与 (n) 在上下文中是同一物理量（粒子数密度）的不同符号写法；公式 5.9 已经给出 (\rho) 的定义。
* (d_q) 没明确写数值时通常按颜色×自旋取 (2N_c)，对 QCD 即为 6。
