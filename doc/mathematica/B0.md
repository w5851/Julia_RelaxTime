# B₀ 积分表达式 - 式 (4.4)

## 论文信息
- **标题**: One Loop Integrals at Finite Temperature and Density
- **作者**: P. Rehberg and S. P. Klevansky
- **编号**: HD-TVP-95-13
- **日期**: September 1995

## 式 (4.4) 完整表达式

两费米子线积分 $B_0$ 的各向异性一阶近似表达式为：

$$
\begin{aligned}
B_0^{(\text{aniso})}(k, m_1, \mu_1, m_2, \mu_2, i\nu_m, \beta, \Lambda, \xi) = 8\pi^2 \int \frac{d^3p}{(2\pi)^3} 
\Bigg[ & \frac{f_{1}^{+\!,\text{aniso}}(p,\theta)}{E_1} \frac{1}{\lambda^2 - 2\lambda E_1 + 2\vec{p}\cdot\vec{k} - k^2 + m_1^2 - m_2^2} \\
& - \frac{f_{1}^{-\!,\text{aniso}}(p,\theta)}{E_1} \frac{1}{\lambda^2 + 2\lambda E_1 + 2\vec{p}\cdot\vec{k} - k^2 + m_1^2 - m_2^2} \\
& + \frac{f_{2}^{+\!,\text{aniso}}(p,\theta)}{E_2} \frac{1}{\lambda^2 + 2\lambda E_2 + 2\vec{p}\cdot\vec{k} - k^2 - m_1^2 + m_2^2} \\
& - \frac{f_{2}^{-\!,\text{aniso}}(p,\theta)}{E_2} \frac{1}{\lambda^2 - 2\lambda E_2 + 2\vec{p}\cdot\vec{k} - k^2 - m_1^2 + m_2^2} \Bigg]
\end{aligned}
$$

## 符号说明

### 基本参数
- $k = |\vec{k}|$：外部动量大小
- $m_1, m_2$：费米子质量
- $\mu_1, \mu_2$：化学势
- $i\nu_m$：玻色子 Matsubara 频率
- $\beta = 1/T$：逆温度
- $\Lambda$：三维动量截断

### 中间变量
- $\lambda = i\nu_m + \mu_1 - \mu_2$（解析延拓后为实数）
- $E_1 = \sqrt{p^2 + m_1^2}$
- $E_2 = \sqrt{p^2 + m_2^2}$
- $\vec{p}\cdot\vec{k} = pk\cos\theta$（在 $\vec{k}$ 沿 z 轴的坐标系中）

### 分布函数
- 假定 $\vec{n}$ 与 $\vec{k}$ 共线（均沿 z 轴），则 $(\vec{n}\cdot\vec{p})^2 = p^2\cos^2\theta$。
- PNJL 各向同性基函数：
	$\displaystyle f_{\Phi}^+(E, \mu) = \frac{\Phi e^{-\beta(E-\mu)} + 2\bar{\Phi} e^{-2\beta(E-\mu)} + e^{-3\beta(E-\mu)}}{1 + 3\Phi e^{-\beta(E-\mu)} + 3\bar{\Phi} e^{-2\beta(E-\mu)} + e^{-3\beta(E-\mu)}}$，
	$\displaystyle f_{\Phi}^-(E, \mu) = \frac{\bar{\Phi} e^{-\beta(E+\mu)} + 2\Phi e^{-2\beta(E+\mu)} + e^{-3\beta(E+\mu)}}{1 + 3\bar{\Phi} e^{-\beta(E+\mu)} + 3\Phi e^{-2\beta(E+\mu)} + e^{-3\beta(E+\mu)}}$。
- 其能量导数写成 $\partial_E f_{\Phi}^{\pm}(E,\mu) = \beta\,\frac{\mathcal{N}_{\pm}'\mathcal{D}_{\pm} - \mathcal{N}_{\pm}\mathcal{D}_{\pm}'}{\mathcal{D}_{\pm}^2}$，其中
	$\mathcal{N}_{+}(x) = \Phi e^{-x} + 2\bar{\Phi} e^{-2x} + e^{-3x}$，$\mathcal{D}_{+}(x) = 1 + 3\Phi e^{-x} + 3\bar{\Phi} e^{-2x} + e^{-3x}$，$x = \beta(E-\mu)$；
	$\mathcal{N}_{-}(y) = \bar{\Phi} e^{-y} + 2\Phi e^{-2y} + e^{-3y}$，$\mathcal{D}_{-}(y) = 1 + 3\bar{\Phi} e^{-y} + 3\Phi e^{-2y} + e^{-3y}$，$y = \beta(E+\mu)$。
- 各向异性一阶修正：
	$f_{1}^{+,\text{aniso}}(p,\theta) = f_{\Phi}^{+}(E_1,\mu_1) + \dfrac{\xi}{2E_1} p^2\cos^2\theta\, \partial_E f_{\Phi}^{+}(E,\mu_1)\big|_{E=E_1}$，
	$f_{1}^{-,\text{aniso}}(p,\theta) = f_{\Phi}^{-}(E_1,\mu_1) + \dfrac{\xi}{2E_1} p^2\cos^2\theta\, \partial_E f_{\Phi}^{-}(E,\mu_1)\big|_{E=E_1}$，
	$f_{2}^{+,\text{aniso}}(p,\theta) = f_{\Phi}^{+}(E_2,\mu_2) + \dfrac{\xi}{2E_2} p^2\cos^2\theta\, \partial_E f_{\Phi}^{+}(E,\mu_2)\big|_{E=E_2}$，
	$f_{2}^{-,\text{aniso}}(p,\theta) = f_{\Phi}^{-}(E_2,\mu_2) + \dfrac{\xi}{2E_2} p^2\cos^2\theta\, \partial_E f_{\Phi}^{-}(E,\mu_2)\big|_{E=E_2}$。
- 当 $\xi \to 0$ 时，$f_{j}^{\pm,\text{aniso}}$ 退化为对应的各向同性 PNJL 分布函数；在 $\Phi,\bar{\Phi}\to 1$ 极限下进一步回到标准费米-狄拉克形式。

## 物理意义

这个表达式描述了在有限温度和化学势下，包含两条费米子线的单圈图积分。该积分在有效场论（如 Nambu-Jona-Lasinio 模型）中用于计算各种物理量，如介子质量和衰变常数。

## 数值计算要点

1. **角向积分**：选择 $\vec{k}$ 沿 z 轴方向，角向积分简化为对 $\cos\theta$ 的积分
2. **截断处理**：动量积分限制在 $|\vec{p}| < \Lambda$ 范围内
3. **解析延拓**：完成 Matsubara 求和后，将 $i\nu_m \to k_0$ 延拓到实轴
4. **奇点处理**：需要小心处理积分中的极点，使用主值积分和 delta 函数
