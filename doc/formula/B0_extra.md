基于公式4.8和公式4.16，我将详细解释条件出现的原因，并给出满足条件时能量 \(E\) 的范围的具体推导过程。

### 公式4.8和公式4.16的回顾
- **公式4.8** 定义了泛函 \(\tilde{B}_{0}^{\pm}\)：
  \[
  \tilde{B}_{0}^{\pm}(\lambda,k,\ m,m^{\prime},\mu,\ \beta,\Lambda)=2\int_{m}^{\Lambda_{E}}dE\,pf(\pm E-\mu)\int_{-1}^{+1}dx\frac{1}{\lambda^{2}+2\lambda E+2pkx-k^{2}+m^{2}-m^{\prime 2}}
  \]
  其中 \(p = \sqrt{E^2 - m^2}\)，\(E\) 是能量变量，积分变量 \(x\) 在 \([-1, 1]\) 范围内。

- **公式4.16** 给出了条件：
  \[
  |\lambda^{2}+2\lambda E-k^{2}+m^{2}-m^{\prime 2}| < 2pk
  \]
  其中 \(\lambda = i\nu_m + \mu_1 - \mu_2\)（在解析延拓后为实数）。

### 条件出现的原因
在公式4.8中，对 \(x\) 的积分被积函数的分母为：
\[
D(x) = \lambda^{2} + 2\lambda E + 2pkx - k^{2} + m^{2} - m^{\prime 2}
\]
这是一个关于 \(x\) 的线性函数。当 \(D(x) = 0\) 时，被积函数出现奇点（极点），即：
\[
x = -\frac{\lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2}}{2pk}
\]
如果这个 \(x\) 值落在积分区间 \([-1, 1]\) 内，则积分会出现奇点。因此，奇点出现的条件为：
\[
-1 \leq -\frac{\lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2}}{2pk} \leq 1
\]
这等价于：
\[
-2pk \leq \lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2} \leq 2pk
\]
即公式4.16的条件：
\[
|\lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2}| < 2pk
\]
因此，该条件确保了在 \(x\) 积分过程中，分母为零的点位于积分路径内，从而导致奇点的出现。

### 满足条件的能量 \(E\) 的范围的推导
为了找到满足公式4.16的 \(E\) 的范围，我们需要解方程：
\[
|\lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2}| = 2k\sqrt{E^2 - m^2}
\]
其中 \(p = \sqrt{E^2 - m^2}\)。这个方程定义了能量 \(E\) 的边界值 \(E_1\) 和 \(E_2\)。

#### 推导步骤：
1. **平方两边以消除绝对值和平方根**：
   \[
   (\lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2})^2 = 4k^2 (E^2 - m^2)
   \]
   因为右边 \(2k\sqrt{E^2 - m^2} \geq 0\)，且平方操作在两边非负时等价于原始方程。

2. **展开并整理为关于 \(E\) 的二次方程**：
   令 \(C = \lambda^{2} - k^{2} + m^{2} - m^{\prime 2}\)，则方程左边为：
   \[
   (2\lambda E + C)^2 = 4\lambda^2 E^2 + 4\lambda C E + C^2
   \]
   方程右边为：
   \[
   4k^2 E^2 - 4k^2 m^2
   \]
   将所有项移到一边：
   \[
   4\lambda^2 E^2 + 4\lambda C E + C^2 - 4k^2 E^2 + 4k^2 m^2 = 0
   \]
   简化：
   \[
   4(\lambda^2 - k^2) E^2 + 4\lambda C E + (C^2 + 4k^2 m^2) = 0
   \]
   除以4：
   \[
   (\lambda^2 - k^2) E^2 + \lambda C E + \frac{C^2 + 4k^2 m^2}{4} = 0
   \]
   这是一个关于 \(E\) 的二次方程。

3. **解二次方程**：
   二次方程的一般形式为 \(a E^2 + b E + c = 0\)，其中：
   - \(a = \lambda^2 - k^2\)
   - \(b = \lambda C\)
   - \(c = \frac{C^2 + 4k^2 m^2}{4}\)
   
   判别式为：
   \[
   \Delta = b^2 - 4ac = (\lambda C)^2 - 4(\lambda^2 - k^2) \cdot \frac{C^2 + 4k^2 m^2}{4}
   \]
   简化：
   \[
   \Delta = \lambda^2 C^2 - (\lambda^2 - k^2)(C^2 + 4k^2 m^2) = k^2 \left[ C^2 - 4m^2 (\lambda^2 - k^2) \right]
   \]
   因此，根为：
   \[
   E = \frac{ -\lambda C \pm \sqrt{\Delta} }{ 2(\lambda^2 - k^2) } = \frac{ -\lambda C \pm k \sqrt{ C^2 - 4m^2 (\lambda^2 - k^2) } }{ 2(\lambda^2 - k^2) }
   \]
   由于 \(\lambda^2 - k^2 = -(k^2 - \lambda^2)\)，可改写为：
   \[
   E = \frac{ \lambda C \mp k \sqrt{ C^2 - 4m^2 (\lambda^2 - k^2) } }{ 2(k^2 - \lambda^2) }
   \]
   代入 \(C = \lambda^{2} - k^{2} + m^{2} - m^{\prime 2}\)，得到最终解（公式4.20）：
   \[
   E_{1/2} = \frac{ \lambda (\lambda^{2} - k^{2} + m^{2} - m^{\prime 2}) }{ 2(k^{2} - \lambda^{2}) } \pm k \frac{ \sqrt{ (\lambda^{2} - k^{2} + m^{2} - m^{\prime 2})^{2} + 4m^{2}(k^{2} - \lambda^{2}) } }{ 2(k^{2} - \lambda^{2}) }
   \]
   这里 \(E_1\) 和 \(E_2\) 是方程的两个解，且假设 \(E_1 < E_2\)。

4. **确定 \(E\) 的范围**：
   方程 \( |\lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2}| = 2k\sqrt{E^2 - m^2} \) 的解 \(E_1\) 和 \(E_2\) 是边界点。

   为了确定满足原始不等式 \( |\lambda^{2} + 2\lambda E - k^{2} + m^{2} - m^{\prime 2}| < 2k\sqrt{E^2 - m^2} \) 的 \(E\) 的范围，我们需要考虑二次项系数 \(a = \lambda^2 - k^2\) 的符号：

   - **情况1：\(a > 0\)（即 \(\lambda^2 > k^2\)）**
     
     此时二次函数开口向上，满足原始不等式的 \(E\) 的范围为：
     \[
     E_1 < E < E_2
     \]
     即能量 \(E\) 介于 \(E_1\) 和 \(E_2\) 之间。

   - **情况2：\(a < 0\)（即 \(\lambda^2 < k^2\)）**
     
     此时二次函数开口向下，满足原始不等式的 \(E\) 的范围为：
     \[
     E < E_1 \quad \text{或} \quad E > E_2
     \]
     即能量 \(E\) 小于 \(E_1\) 或大于 \(E_2\)。

   - **情况3：\(a = 0\)（即 \(\lambda^2 = k^2\)）**
     
     此时二次方程退化为线性方程，需要单独处理。这种情况下，原始不等式变为：
     \[
     |2\lambda E + C| < 2k\sqrt{E^2 - m^2}
     \]
     其中 \(C = m^2 - m^{\prime 2}\)。这种情况较为特殊，在实际物理问题中较少出现。

   在实际计算中，\(E\) 还必须满足积分区间 \([m, \Lambda_E]\)，其中 \(\Lambda_E = \sqrt{\Lambda^2 + m^2}\)。因此，实际有效的范围为上述解与 \([m, \Lambda_E]\) 的交集。

   例如，对于 \(a > 0\) 的情况，有效范围为：
   \[
   E \in [\max(m, E_1), \min(\Lambda_E, E_2)]
   \]
   对于 \(a < 0\) 的情况，有效范围为：
   \[
   E \in [m, \min(\Lambda_E, E_1)] \cup [\max(m, E_2), \Lambda_E]
   \]
   如果 \(E_1\) 或 \(E_2\) 不在积分区间内，则奇点可能只出现在部分区间或不存在。

#### 关于二次项系数的说明
在二次方程中，二次项系数 \(a = \lambda^2 - k^2\) 的符号确实会影响不等式的解集。在PDF的上下文中，作者没有明确假设 \(\lambda^2 - k^2 > 0\)，但在数值实现中必须考虑所有情况。

在数值计算中，程序会先计算 \(E_1\) 和 \(E_2\)，然后根据 \(a\) 的符号和 \(E_1\)、\(E_2\) 是否在 \([m, \Lambda_E]\) 内来确定积分范围。如PDF第10页所述的三种情况需要根据 \(a\) 的符号进行相应调整。

### 总结
- 公式4.16的条件确保公式4.8中对 \(x\) 的积分出现奇点。
- 满足条件的能量 \(E\) 的范围由公式4.20定义的 \(E_1\) 和 \(E_2\) 以及二次项系数 \(a = \lambda^2 - k^2\) 的符号共同决定：
  - 当 \(a > 0\) 时，\(E_1 < E < E_2\)
  - 当 \(a < 0\) 时，\(E < E_1\) 或 \(E > E_2\)
- 实际数值积分中，虚部的积分范围需要根据上述解与积分区间 \([m, \Lambda_E]\) 的交集确定，并根据 \(E_1\) 和 \(E_2\) 的位置调整。
